# Base120 Governance

## Version Policy

- v1.0.x is frozen.
- Changes permitted:
  - Security fixes
  - CI hardening
  - Documentation clarifications
- No semantic, schema, or registry changes.

## Authority

- Git tags define canonical releases.
- main reflects the latest stable state.
- All changes require CODEOWNER approval.

## Escalation

Failure modes FM30+ require governance review.

## Future Work

- v1.1.0 introduces signed artifacts and extended mappings.
